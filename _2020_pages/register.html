Application

The deadline is Friday August 29th 2020 (AoE).

To register, send an email to school@pymor.org:

  • with an attached short motivation letter.

Short motivation letter

Please state in your motivation letter:

  • your background (mathematics, engineering, partial differential equations, systems and control theory, ...),
  • previous model order reduction knowledge (reduced basis methods, system-theoretic methods, ...),
  • your motivation for using pyMOR,
  • optionally, the problem to which you would like to apply model order reduction.

This information will only be used to adapt the School's program.
